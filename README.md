# CSA — Comunidade que Sustenta a Agricultura

Plataforma web agrícola que conecta produtores rurais e consumidores conscientes.

## Páginas

- **Início** — Homepage com hero, categorias, produtos, cursos e receitas
- **Marketplace** — Loja de produtos agrícolas (em desenvolvimento)
- **Cursos** — Cursos de agricultura sustentável (em desenvolvimento)
- **Receitas** — Receitas da comunidade (em desenvolvimento)
- **Integrantes** — Membros e produtores (em desenvolvimento)
- **Perfil** — Área do usuário (em desenvolvimento)
- **Login** — Autenticação (em desenvolvimento)
- **Design System** — Documentação completa de cores, tipografia, ícones e componentes

## Tecnologias

- HTML5, CSS3, JavaScript (vanilla)
- Noto Sans (Google Fonts)
- 35+ ícones SVG inline customizados
- Responsivo (mobile-first)
